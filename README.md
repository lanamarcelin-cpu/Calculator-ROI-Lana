# Calculator-ROI-Calculator
This project estimates the **10-year return on investment (ROI)** of college majors based on tuition costs and projected salary growth.
Built using **Excel** with dynamic formulas, conditional formatting, and data-driven visuals. Optionally visualized with **Tableau**.

---

## Files Included
'College_ROI_Calculator_Project.csv - The interactive Excel workbook 
'Visuals for Calculator ROI Project

---

## Project Objectives 
Help students assess the long-term ROI of different majors
Practice Excel modeling skills using compound interest, formulas, and visualization
Prepare data for visualization tools like Tableau or Power BI

---

## Tools Used
**Microsoft Excel**: Calculations, formulas, conditional formatting 
**Tableau**: Data visualizations 

---

## What I learned
Creating dynamic formulas (IF, VLOOKUP, compound growth)
Using Excel for decision-supports modeling
Visualizing insights through bar charts, and dashboards

## Inspiration
Inspired by common questions student face when selecting a college major. This model helps simulate the financial impact of those decisions.
